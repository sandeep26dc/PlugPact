## 🛠️ Build & Architecture
Developed by **SANDEEP SOM**, PlugPact utilizes a modern tech stack to ensure zero-crash stability and OLED safety.

*   **Kotlin 2.0 / Jetpack Compose** - For the glossy, premium interface.
*   **AGSL Runtime Shaders** - For the continuous 3D electric spark effect.
*   **Foreground Hardware Monitor** - Real-time mA/mV telemetry.
*   **OLED Anti-Burn-In Engine** - Autonomous pixel shifting logic.
